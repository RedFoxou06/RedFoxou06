## Projets

**[FoxHub](https://foxhub.redfoxou.dev)** — Store public centralisant la distribution de mes outils. Développé avec Ionic React et Capacitor, il permet d'installer mes APK ou de lancer mes outils web via une interface unique synchronisée en temps réel.  
`HTML` `CSS` `JavaScript` `Ionic React` `Capacitor`

---

**Guardian** — Application Android qui teste en direct la résistance de tes mots de passe face aux attaques. 100% offline : aucune donnée n'est envoyée, tout l'audit se fait sur l'appareil.  
`Ionic React` `Capacitor`

---

**[RedSafe](https://foxhub.redfoxou.dev)** — Gestionnaire de mots de passe mobile axé sur la confidentialité absolue. Zéro serveur : les données sont stockées localement en fichiers JSON via le système de fichiers natif. Générateur de clés complexes intégré.  
`Python` `Ionic React` · [GitHub](https://github.com/RedFoxou06/gestionnaireMDP)

---

**[VALOTRACK](https://tracker.redfoxou.dev)** — Tracker de stats compétitives Valorant en temps réel. Rang, K/D, win rate, headshot %, série en cours. Déployé sur serveur Apache avec proxy PHP pour sécuriser la clé API.  
`HTML` `CSS` `JavaScript` `PHP` `Apache`
